# Google URL Update Tools

Small Google Apps Script tools for updating links in Google Sheets and Google Slides.

## What This Does

This toolkit has two scripts:

- `scripts/sheets-url-builder.gs`: runs inside a Google Sheet and fills Column B with updated URLs.
- `scripts/slides-url-updater.gs`: runs inside a Google Slides deck and updates embedded slide links using the mapping table from the Google Sheet.

## Sheet Layout

Use a Google Sheet with this structure:

| Column | Header | Purpose |
| --- | --- | --- |
| A | CurrentURL | The URL currently embedded in Google Slides |
| B | NewURL | The new URL generated by the Sheet script |
| C | Text to find | The part of the URL that needs to change |
| D | Replace with | The replacement text |

## Basic Workflow

1. Paste current URLs into Column A.
2. Add replacement rules in Columns C and D.
3. Run the Sheet script to fill Column B.
4. Open the Google Slides deck.
5. Run `URL Tools > Update slide links`.

## Privacy Note

Do not commit real email addresses, private URLs, or real mailto drafts to a public GitHub repository.

In `slides-url-updater.gs`, replace this placeholder after copying the script into Google Slides:

```javascript
const URL_MAPPING_SPREADSHEET_ID = 'PASTE_YOUR_SHEET_ID_HERE';
```

Use your real Google Sheet ID only inside your private Google Slides Apps Script project.
